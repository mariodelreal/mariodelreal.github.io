TITLE DESCRIPTION
NES IDE A web app that allows for easier development and testing of NES video games. It features a tile, palette, and code editor with a project based system to build programs. The application leverages the NESASM assembler to create the game binary and FCEUX to emulate the created game. The web browser UI is served through a local NodeJS server which also handles communications between the UI and desktop when creating, opening, saving, and running projects.
Enhanced Reading This Google Chrome extension puts an end annoying websites preventing you from reading an article through auto-playing videos or ads. When the extension is enabled, it allows the user to select a container on a website and have its text extracted into another tab; allowing the user to leave the website that is slowing down their browser or playing audio. The user can change the text's font, size, color and the text's background color to make reading it easier. The user can also "speed read" the text through an implementation of a speed reading technique similar to Spritz's.
Algorithmm for Trading the Soybean Crush Spread Using Strategy Studio from RCM-X in Chicago, this algorithmm uses market trends to determine when it should buy or short contracts for the Soybean Crush Spread. Using the API developed by Strategy Studio, a dll file is created to operate with the Strategy Studio desktop application. The algorithm uses implementation defined changeable variables to hone in which parameters will yield better returns in current market conditions.
